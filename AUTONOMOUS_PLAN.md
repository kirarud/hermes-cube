# Hermes Cube — Автономный план развития

**Старт:** 25 июня 2026
**Контекст API:** deepseek-v4-flash (1M), OpenModel.ai, лимит ~3-4 RPM
**Подход:** Step-by-step, sub-агенты (макс 2 параллельно), документация в Obsidian после каждого этапа

## Сквозные принципы
- Каждый этап = коммит в git + запись в Obsidian
- Новые идеи внедряются по ходу, документируются
- Ничего не ломать — тест синтаксиса после каждого изменения
- API-лимиты: пауза 5-10с между batches, максимум 3 быстрых запроса
