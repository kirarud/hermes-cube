# 📝 TextOverlay — Полноэкранный слой для букв

## Архитектура

Отдельное `tk.Toplevel` окно:
- Размер: весь экран (screenwidth × screenheight)
- Прозрачный фон (transparentcolor #000001)
- overrideredirect (без рамки)
- Скрыт (withdraw) когда нет контента
- Показывается (deiconify) при spawn_text_particles или _spawn_agent

## Как работает spawn_text_particles

1. Текст разбивается на строки (wrap по ширине экрана)
2. Каждая строка центрируется под центром экрана
3. Для каждой буквы:
   - Стартовая позиция = координаты частицы куба (в screen-space)
   - Целевая позиция = позиция в строке
   - Каждая буква — canvas.create_text на text_canvas
4. Анимация — 3 фазы:
   - Fly: ease-out cubic за 40 кадров от куба к строке
   - Glow: свечение 10 кадров, затем удержание
   - Fade: затухание за 50 кадров до конца max_life
