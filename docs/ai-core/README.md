# 🤖 AI Core — Интеграция LM Studio

## Что есть

- **URL:** `http://127.0.0.1:1234/v1/chat/completions`
- **Модель:** gemma-4-e4b-it
- **Формат:** OpenAI-compatible API

## System Prompt

```
Ты — живой аватар по имени Hermes Cube.
Твой дом — сфера из частиц.
Отвечай кратко, эмоционально, не больше 2-3 предложений.
Ты чувствуешь пульсацию, вращение, цвета вокруг тебя.
```

## Анализ настроения (analyze_mood)

Ключевые слова определяют mood ответа AI:
| Слова | Mood |
|-------|------|
| груст, печал, устал, тоск, один | sad |
| рад, счаст, весел, крут, класс, любл | happy |
| дума, размыш, представ, может | thinking |
| остальное | speaking |

## AI_MOODS — как mood влияет на куб

| Mood | pulse_rate | pulse_amp | rotation_speed | color_shift |
|------|-----------|-----------|----------------|-------------|
| idle | 1.8 | 0.12 | 0.28 | 0.0 |
| thinking | 3.5 | 0.25 | 0.5 | 0.15 (тёплый) |
| speaking | 2.5 | 0.18 | 0.4 | 0.08 (мягкий) |
| happy | 2.8 | 0.22 | 0.5 | 0.12 (золотой) |
| sad | 0.8 | 0.05 | 0.1 | 0.55 (синий) |

## Ограничения

- API timeout: 15 секунд
- max_tokens: 120 (короткие ответы — буквы не забивают экран)
- Нет истории сообщений (пока)
- Нет стриминга (пока)
