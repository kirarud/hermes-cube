# 🔤 Symbol Cube — Куб из символов

## Концепция

Каждая частица куба — не точка, а буква/символ. Поверхность куба — текст, который может меняться.

Сейчас: куб из цветных квадратиков (grid of rects)
Будет: куб из букв (grid of chars) — каждая буква может быть частью слова на грани

## План

1. char_matrix = np.chararray((N, N)) в CubeEngine
2. Режим отображения: create_text вместо create_rectangle
3. AI может "писать" на гранях куба
4. Анимация: символы затухают/заменяются
