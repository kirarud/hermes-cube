# 📚 Obsidian Graph Cube — Куб из заметок

## Концепция

Второй невидимый куб, где каждая частица — заметка из Obsidian Vault.
Граф знаний растёт от центра куба (Z=0) к поверхности (Z=1).

## Архитектура (проект)

1. Парсер Obsidian Vault (frontmatter + wikilinks)
2. Построение графа (nodes = notes, edges = [[links]])
3. Mapping на сферу: Z = важность/давность
4. Визуализация: второй прозрачный overlay
5. Клик → открыть в Obsidian
