# 🧬 Agents — Живые частицы-агенты

## Концепция

Каждая частица куба — потенциальный агент. Агент может:
- Вылететь из куба и парить в пространстве
- Рисовать пиксельные спрайты (смайлики, курсоры, иконки)
- Создавать интерфейсы (кнопки, ползунки)
- Множиться (спавнить дочерних агентов)

## Текущее состояние (прототип)

### Клавиша A — спавн агента

1. Выбирается случайная частица куба
2. Создаётся пиксельный спрайт (7×10 пикселей)
3. Агент летит к случайной точке экрана (ease)
4. Зависает с bobbing-анимацией (синусоида 3px)
5. Пульсирует цветом (glow)
6. Через ~5 секунд затухает и исчезает

### Спрайты

```python
AGENT_SPRITES = {
    'smiley':  # 😊 — 10×7 пикселей
    'cursor':  # 👆 — 7×6 пикселей
    'star':    # ⭐ — 7×7 пикселей
}

AGENT_PALETTE = [
    0: прозрачный
    1: #e94560 (красный куба)
    2: #f59e0b (золотой)
    3: #22c55e (зелёный)
    4: #6366f1 (индиго)
    5: #ec4899 (розовый)
    6: #0ea5e9 (голубой)
]
```

### Структура агента

```python
agent = {
    'x', 'y':      # текущая позиция (screen-space)
    'tx', 'ty':    # целевая позиция
    'life', 'max_life':  # таймер жизни
    'sprite':      # numpy matrix пикселей
    'items': []    # canvas item ID для каждого пикселя
    'phase':       # fly → hover → vanish
    'arrived':     # флаг прибытия
}
```
